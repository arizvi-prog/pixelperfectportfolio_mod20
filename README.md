Overview
This portfolio is built using React and is designed to highlight my work and skills as a web developer. The application includes sections for About Me, Portfolio, Contact, and Resume. Each section is accessible via a navigation menu that allows for smooth, single-page navigation without page reloads.

Features
Header: Contains my name and navigation links to different sections of the portfolio.
Navigation: Provides links to About Me, Portfolio, Contact, and Resume sections.
About Me: Displays a photo or avatar and a short bio.
Portfolio: Showcases six projects with images, titles, and links to both deployed applications and GitHub repositories.
Contact: Includes a form for visitors to get in touch, with validation for required fields and email format.
Resume: Provides a downloadable resume and lists my proficiencies.
Footer: Contains links to my GitHub, LinkedIn, and other professional profiles.
Getting Started
To run this project locally, follow these steps:

Clone the repository:
git clone https://github.com/arizvi-prog/pixelperfectportfolio_mod20.git
cd react-portfolio
npm install
npm run dev
Open your browser and go to http://localhost:3000 to view the application.


Technologies Used
React: For building the user interface.
Vite: For a fast development environment and build tool.
SCSS: For styling with advanced features.
React Router DOM: For navigation between sections.
Netlify: For deploying the application.
